# Django Weekend Assignment – TechRise
## Overview
This project contains 3 apps: ToDo, Quotes, and Contact.
## How to Run
1. Clone the repo
2. Create a virtual environment
3. Run `pip install -r requirements.txt`
4. Run `python manage.py runserver`
## Features
- Task manager (CRUD)
- Motivational quotes
- Contact page
